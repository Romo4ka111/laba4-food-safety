# Макет интерфейса

## Страница входа `/login`

- заголовок «Вход в аккаунт»;
- поле логина/email;
- поле пароля;
- кнопка входа;
- сообщение об ошибке при неверных данных;
- подсказка с тестовыми пользователями.

## Страница регистрации `/register`

- поле имени пользователя;
- поле email;
- поле пароля;
- поле повтора пароля;
- кнопка регистрации;
- проверка сложности пароля;
- сообщение об успешной регистрации.

## Главная страница `/`

- шапка с меню;
- улучшенный спинер с направлениями контроля;
- статистика по проверкам;
- фильтры поиска, риска и статуса;
- карточки проверок;
- кнопки «Подробнее», «Редактировать», «Удалить»;
- пагинация.

## Карточка проверки `/detail/:id`

- основные поля проверки;
- статус и уровень риска;
- описание проверки;
- ответственный пользователь;
- кнопки возврата и редактирования.

## Административный раздел `/admin/users`

- таблица пользователей;
- изменение роли;
- изменение статуса;
- доступ только для администратора.

## Журнал действий `/audit-log`

- таблица событий;
- время;
- пользователь;
- действие;
- объект;
- IP-адрес;
- доступ только для администратора.
